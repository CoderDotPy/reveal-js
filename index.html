Code Sample (Python)


from datetime import datetime

def npv(rate, cashflows):
    """Net Present Value given a discount rate and a list of cashflows."""
    return sum(cf / ((1 + rate) ** t) for t, cf in enumerate(cashflows))

r = 0.1
cfs = [-1000, 400, 400, 400, 400]
print(f"NPV@{r:.0%} = {npv(r, cfs):.2f}")

Financial Math

Compound amount:

$$A = P\,\left(1 + \frac{r}{n}\right)^{nt}$$

Net Present Value (NPV):

$$\mathrm{NPV} = \sum_{t=0}^{T} \frac{C_t}{(1+r)^{t}}$$

Internal Rate of Return (IRR): the rate \(r\) such that \(\mathrm{NPV}(r)=0\).

Connect these formulas to the earlier code: the Python example computes NPV numerically for a fixed rate.
